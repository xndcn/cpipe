# ISP Golden Fixtures

These EXR pairs are deterministic ISP node goldens generated by:

```bash
cmake --build --preset linux-debug --target gen_golden_isp_nodes
build/linux-debug/tests/gen_golden_isp_nodes tests/golden
```

They are self-referenced fixtures for current node math unless a per-fixture
`reference.md` states otherwise. See `docs/phase-01-walking-skeleton.md` PD-69
and `docs/phase-02-classic-nodes-hdr.md`.
