`pixel8pro-qbc.dng` is a 32x32 synthetic Sony-style 4x4 Quad Bayer DNG
stand-in generated by `tests/unit/dng_test_fixture.hpp`.

It is intentionally not a real Pixel 8 Pro 50 MP Pro-mode capture. T21 keeps the
stable corpus path required by the P2 smoke command while P2-R4 continues to
track procurement of a licensed real Pixel / alternate-phone QBC DNG. The
fixture exercises the DNG reader's 4x4 CFA metadata path and the
`demosaic.quad_bayer_remosaic -> Bayer pipeline -> HEIF SDR` no-crash smoke.
